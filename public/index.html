Hacker News stuff

All my random Hacker News hacks in one place -

  • Hacker News Daily Api: REST API built with TypeScript and Express for HN Daily.
  • StumbleUpon Hacker News: StumbleUpon but you get Best Stories from Hacker News. Built using NextJS 13 and TailwindCSS.
  • Random Hacker News: Teleport to the front page of Hacker News on a random date! Made using VueJS, TypeScript, TailwindCSS.
